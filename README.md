# 👀 眼睛小勇士 - 儿童弱视训练游戏广场

通过有趣的互动小游戏，帮助小朋友进行弱视康复训练。每天坚持15-20分钟，让训练变得快乐！

## 🎮 游戏列表

| 游戏 | 训练目标 | 说明 |
|------|----------|------|
| 🔍 找不同 | 视觉注意力 | 对比两张图片，找出不同之处，共3个关卡 |
| 🌈 颜色配对 | 颜色辨识力 | 翻开卡片记忆颜色，找到相同颜色配对 |
| ⭐ 追踪星星 | 眼球追踪能力 | 跟随星星移动轨迹（圆形、8字、方形、星形） |
| 🔷 形状匹配 | 形状辨识力 | 找到相邻相同形状并消除 |

## 🚀 快速开始

```bash
# 安装依赖
npm install

# 启动开发服务器
npm run dev

# 构建生产版本
npm run build
```

## 🛠 技术栈

- **React 18** + **TypeScript**
- **Vite** 构建工具
- **Tailwind CSS** 样式框架
- **Lucide React** 图标库

## 🌐 在线体验

## 💡 使用建议

- 建议每天训练 **15-20分钟**
- 保持屏幕亮度适中，避免强光环境
- 训练时保持正确坐姿，眼睛距屏幕 **30-40cm**
- 如有不适请立即停止，遵医嘱进行训练
